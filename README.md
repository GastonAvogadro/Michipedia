# Sitio web Michipedia

![michipedia](http://imgfz.com/i/YlRkfTs.jpeg)

Sitio web desarrollado para poder ver y buscar las distintas razas de gatos que existen.

### Link al sitio
### https://michipedia.vercel.app/

### Instalación y ejecución

```sh
npm install
npm run dev
```

### Tech: 

- React
- Redux
- Tailwind
- Axios

### Librerias:

- Framer Motion
